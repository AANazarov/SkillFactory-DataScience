### ***Project Description / Описание проекта***
Creating a Python program that guesses a number in the minimum number of attempts.  

Создание программы на Python, которая угадывает число за минимальное число попыток.
***  

### ***Task description / Описание задачи***
The program guesses a randomly generated number using a half division algorithm.  
The program also processes statistical data on guessing numbers: calculates the average number of attempts and builds a histogram of the distribution of the number of attempts.  
  
Программа угадывает случайно сгенерированное число с использованием алгоритма половинного деления.  
Также программа обрабатывает статистические данных по угадыванию чисел: рассчитывает среднее число попыток и строит гистограмму распределения числа попыток.  
***
